Test TLDs
ICANN created a set of top-level Internationalized domain names in October 2007 for the purpose of testing the use of IDNA in the root zone and within those domains. They are expected to be available for a limited evaluation period.[11][12]. Each of these TLDs encodes a word meaning "test" in the respective language.[12][13]
Each of these domains contains only one site with the word example encoded in the respective script and language. These example.test sites are test wikis used by ICANN.
