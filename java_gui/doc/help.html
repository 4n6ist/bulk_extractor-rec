Overview



The Bulk Extractor Viewer User Interface allows the user to browse through Features in Feature files created using the bulk_extractor utility. The Bulk Extractor Viewer allows browsing through Features of multiple reports. Browsed Features show their corresponding content in the Image view. Feature lists may be bookmarked and exported.


Bulk Extractor Viewer functionality includes the following:

  • Feature reports extracted by bulk_extractor are browsed using the Bulk Extractor Viewer User Interface.

  • The bulk_extractor feature extraction tool may be launched from the Bulk Extractor Viewer User Interface.

  • Multiple feature reports may be opened at once.

  • A Feature History allows quick access to previously selected Features.

  • Multiple features may be bookmarked. Bookmarked features may be exported to a file along with the section of their image data.

  • Feature and Image text may also be copied from the screen to the system clipboard.

  • Settings (opened Reports, bookmarks, and configuration settings) are persistent and may be exported and imported on another system.

  • Feature and Image content may be highlighted to highlight feature text or specific user text.

  • The Image view may be displayed in text or binary hexadecimal format and may be paged forward and backwards from a browsed Feature to view additional context.

  • The Image view may be paged forward and backwards.

  • Some meta data associated with Image media may be viewed.

Interface

The interface consists of a Reports area, a Features area, and a Navigation area.


Reports

Opened Reports and their associated Feature files appear in the Reports listing at the left. Open reports so that their features may be browsed through by using menu File | Open Report. Click on a feature file to view its features in the Feature File column in the middle. Close reports using File | close Report.


Features

Click on a feature file in the Reports listing to view the features in that file. Click on a Histogram file in the Reports listing to view the Histogram listing of the Feature File and also the Referenced Feature File associated with the selected Histogram file. Click on a Feature line to select that Feature.


Selecting a Histogram line shows all matching Features in the Referenced Feature File listing. Selecting a Feature navigates to that Feature.


To copy Feature text to the system clipboard, select text by dragging the cursor over the desired text, then use menu Edit | Copy.


Feature Filter

The set of features shown in the Features File listing can be filtered by entering filter text. Filtering can be set to ignore or match capitalization.


Navigation

Selecting a feature path in the Features listing navigates to the selected feature. When a feature is navigated to, its feature attributes are shown at the top right and the image view corresponding to the feature are shown in the lower right.


A Navigation history is kept of Features that have been navigated to. Select a Feature from the Navigation history to navigate to these Features.


The feature that is navigated to may be bookmarked via the bookmark button or deleted from the Navigation history.


Highlight

Text in the Feature listings and in the Image view may be highlighted. Click the arrow up button to highlight the feature text that is navigated to. Click the arrow right button to highlight user text that is typed into the input field to the right of the button. User text highlighting can be set to ignore or match capitalization.


Image

The Image area shows a window of the image surrounding the Feature that is currently navigated to. The image view may be shown in Text or Hex format. The image window may be scrolled forward or backward from the navigated Feature to see additional context.


To copy Image text to the system clipboard, select text by dragging the cursor over the desired text, then use menu Edit | Copy.


Menus

  • Use the File menu to open and close reports, export bookmarks, and manage Cases.

  • Use the Edit menu to copy selected features and image data to the system clipboard and to clear the Navigation history.

  • Use the View menu to manage view settings and to observe properties.

  • Use the Help menu to obtain version information and help and to manage runtime logs and diagnostics.

Examples

The following example creates Feature files, opens a Report for the Feature files, selects a Feature file, then examines a Feature from the selected Feature file:


Feature Extraction

Scan a media image to generate a new directory of Feature files. Use menu Tools | Run bulk_extractor to begin the scan. Specify the Image file and the new directory to contain the feature files. The new directory must not exist yet. Set other options, as desired. The scan takes time on large Image files. A progress indicator shows when the scan is done.


Opening a Report

Open a report generated by bulk_extractor using menu File | Open Report. Specify the directory of the Feature files. For the path of the Image file, use the path from the directory unless the image file has been moved.


Navigating to a Feature

Select a Feature file from an opened report. Click on the expander icon next to the image file name in the Reports area in order to view the list of generated Feature files associated with the image. Then click on the Feature file of interest. Select a Feature. In the Feature listing, click on a Feature entry to navigate to it.


Exporting Bookmarks

Create some bookmarks by navigating to various Features and bookmarking them using the bookmark button. Use menu File | Export Bookmarks to open the Export Bookmarks window. Adjust the Bookmarks list as desired, enter the Bookmarks Filename, then click the Export button.

Reporting Errors

If anything fails, please report it. Please provide the following information:

  • Version information

    • The version number of BEViewer, obtained from menu command Help | About Bulk Extractor Viewer.

    • The Download used to install Bulk Extractor. For Windows, this is typically Windows Installer be_installer.msi. Otherwise, please indicate the bulk_extractor version used and how it was installed or that it is not installed.

    • The Operating System you are using, typically Windows, Linux, or Macintosh.

  • Description

    • Observed behavior.

    • Expected behavior.

    • Context such as inputs leading up to the error.

    • If helpful, please also include a screen snapshot and a description of what should be showing.

  • Runtime Log

    • Please include a copy of the runtime log. To obtain the runtime log, please copy it using menu command Help | Log Report | Copy to System Clipboard, and then paste into your report. If needed, you may delete all but the relevant bottom portion of the log report.
